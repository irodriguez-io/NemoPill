# Glossary

## How To Use This File

- This file is optional. Use it once the Ubiquitous Language table in `01_product_vision_and_scope.md` outgrows roughly 15 terms.
- Definitions here must match `01_product_vision_and_scope.md` exactly. If the two ever drift, `01` wins and this file is updated.
- Every term used in `_context/` files, code, UI copy, and stakeholder communications should appear here once the project matures.
- Group terms by domain area to make scanning easier. Within each group, list terms alphabetically.

## Conventions

- Each term has one canonical form (the entry name) and an explicit list of synonyms or legacy terms to avoid.
- Definitions describe the business meaning, not the implementation.
- If a term has both a business meaning and a technical meaning, document both and label them.
